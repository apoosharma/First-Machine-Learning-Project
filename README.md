# Energy Consumption Prediction & Anomaly Detection

## 📌 Overview
This project predicts energy consumption and identifies anomalies using machine learning.

## 📊 Dataset
- Contains information about teams, athletes, and medals

## ⚙️ Technologies Used
- Python
- Pandas
- Scikit-learn
- numpy
- Seaborn

## 🚀 Workflow
1. Data cleaning
2. Handling missing values
3. Train-test split
4. Model training
5. Prediction
6. Evaluation using MAE

## 📈 Results
Achieved good prediction performance using linear regression.
