# Energy Consumption Prediction & Anomaly Detection

## 📌 Overview
This project predicts energy consumption and detects anomalies using machine learning.

## 📊 Dataset
The dataset contains information about teams, athletes, and medals.

## ⚙️ Technologies Used
- Python
- Pandas
- Scikit-learn
- Seaborn

## 🚀 Workflow
1. Data cleaning
2. Handling missing values
3. Train-test split
4. Model training
5. Prediction
6. Evaluation using MAE

## 📈 Results
The model predicts values with good accuracy and low mean absolute error.

## ▶️ How to Run
1. Install dependencies:
   pip install -r requirements.txt
2. Open Jupyter Notebook
3. Run all cells
